Make it Obvious

If a user doesn't know what a button does, they may be afraid to click on it. User interfaces shouldn't be scary to use.

Example (Bad)
It's hard to tell what this button will do. The wording isn't descriptive, and there isn't enough context to tell what it does.
Example (Good)
We can tell exactly what this button will do at a glance. Extra information is available if you hover over it, too!
