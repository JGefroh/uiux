Give Users Feedback

When a user clicks on a button, they expect something to happen. If you don't give them feedback, they may get confused.

Example (Bad)
A button without any feedback. Are you sure you clicked it?
Example (Good)
A button that is disabled when clicked. Good feedback, but how do you tell if the application froze?
Example (Great)
A button that is disabled and has a spinner next to it when clicked. Now you feel like the system is doing something.

The button can have a spinner inside and describe what is happening, too.
