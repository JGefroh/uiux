jgTree

jgTree is a basic tree that allows you to select children from a hierarchy.


It features the following:
  • 2-way binding with its data source - the tree updates as the source updates.
  • Collapsing/Expanding nodes and selecting nodes are two separate operations.
  • Works with your data, instead of having to change the data to work with the directive.
  • Customizable icons, CSS, and animation.
  • Supports up to 7 levels of nesting.
  • Several convenience watchers.
