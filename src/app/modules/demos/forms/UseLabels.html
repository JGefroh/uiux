Use Labels

Input fields should be clearly marked and identified. There's been an unfortunate trend to put the label inside the field itself. I recommend against it.

Example (Bad)

Having the label inside the input field makes it confusing because it disappears once a user starts typing.

Did you forget what format the account number should've been in? Don't force users to remember how to enter data.

Example (Good)

Having the label always visible lets you look at it if you ever forget.

Users don't have to remember anything - they can always see what is expected of them.
